# SQL

This folder contains SQL practice covering two semesters of database coursework:

- **2nd Semester** — Database Management Systems (DBMS)
- **4th Semester** — Advanced Database Management Systems (ADBMS)

All queries are written in **MySQL** and tested against the `hr` schema
and custom tables used in lab exercises.

## Folder Structure

| Folder | Concepts Covered |
|--------|-----------------|
| 01_basics | SELECT, WHERE, ORDER BY, LIMIT, aliases, aggregate functions |
| 02_joins | INNER, LEFT, RIGHT, FULL OUTER, CROSS, SELF, NATURAL JOIN |
| 03_functions | String, date, and conversion functions |
| 04_aggregate_groupby | GROUP BY, HAVING, DISTINCT, REGEX, COUNT DISTINCT |
| 05_subqueries | Nested, correlated, EXISTS, ANY, ALL, scalar subqueries |
| 06_stored_procedures | Procedures, IN/OUT/INOUT parameters, IF logic, loops |
| 07_triggers | BEFORE/AFTER INSERT/UPDATE/DELETE, audit logging |
| 08_transactions | COMMIT, ROLLBACK, SAVEPOINT, isolation levels, locking |
| 09_window_functions | RANK, DENSE_RANK, LAG, LEAD, running totals, EXPLAIN |
| 10_views_indexes | Views, updatable views, indexes, unique indexes |



## Semester Mapping

| Folder | Semester |
|--------|----------|
| 01_basics → 05_subqueries | 2nd Semester (DBMS) |
| 05_subqueries (advanced) | Mixed — both semesters |
| 06_stored_procedures → 10_views_indexes | 4th Semester (ADBMS) |
